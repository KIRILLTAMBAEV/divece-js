Документация о синтаксисе языка

О языке

Markdown — это язык разметки, в котором, структура страницы описывается специальными символами. Эти символы визуально пытаются отразить графическое оформление элементов верстки.

Взаимодействие с HTML

В языке Markdown можно вставлять любые тэги на языке HTML. Большинство конверторов, смогут обработать их вместе с языком Markdown.

Заголовки

Для создания заголовка первого уровня в языке Markdown, после заголовка нужно добавить символы знака равенства =.

Для создания заголовка второго уровня в языке Markdown, после заголовка нужно добавить символы тире -.

Для задачи заголовка любого уровня в языке Markdown можно добавить определенного количество знаков # перед заголовком.

Пример:

Заголовок первого уровня ======================== Заголовок второго уровня ------------------------ # Заголовок первого ## Второго ###### ...шестого уровня

Приведет к генерации следующего HTML:

Заголовок первого уровня

Заголовок второго уровня

Заголовок первого

второго

...шестого уровня

Акцент и значимость текста

Для указания акцента на слово, фразу или предложение необходимо обернуть текст в символы _ или *.

Для указания высокой важности текста, текст следует обернуть в два символа _ или *.

Пример:

*текст на который установлен акцент* _текст на который установлен акцент_ **текст высокой важности** __текст высокой важности__

Приведет к генерации следующего HTML:

текст на который установлен акцент
текст на который установлен акцент

текст высокой важности
текст высокой важности

Ссылки

Для создания ссылки в языке Markdown необходимо использовать следующий синтаксис.

[Пример ссылки](https://example.com) [Пример ссылки c заголовком](https://example.com "Заголовок ссылки") <https://example.com>

Приведет к генерации следующего HTML:

Пример ссылки
Пример ссылки c заголовком
https://example.com

Списки

Неупорядоченные списки

Для создания неупорядоченного списка, перед элементами списка нужно добавить символ *, + или -.

Пример:
* первый элемент списка * второй элемент списка * третий элемент списка + первый элемент списка + второй элемент списка + третий элемент списка - первый элемент списка - второй элемент списка - третий элемент списка

Приведет к генерации следующего HTML:

  • первый элемент списка
  • второй элемент списка
  • третий элемент списка
  • первый элемент списка
  • второй элемент списка
  • третий элемент списка
  • первый элемент списка
  • второй элемент списка
  • третий элемент списка

Упорядоченные списки

Для создания упорядоченного списка, перед элементами списка нужно поставить номер элемента списка.

Пример:
1. первый элемент списка 2. второй элемент списка 3. третий элемент списка

Приведет генерации следующего HTML:

  1. первый элемент списка
  2. второй элемент списка
  3. третий элемент списка

Списки определений

Для создания списков определений необходимо использовать следующий синтаксис.

слово : определение слова другое слово : другое определение слова

Приведет к генерации следующего HTML:

слово
определение слова
другое слово
другое определение слова

Цитаты

Как в почтовых сообщениях цитирование текста производится символами > в начале каждой строки.

Пример:

> Lorem ipsum, dolor sit amet consectetur > adipisicing elit. Fugiat esse eius reprehenderit > quasi totam velit! Voluptas illo hic incidunt nemo > numquam est dicta pariatur alias optio, totam eius > amet perferendis?

Приведет к генерации следующего HTML:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Fugiat esse eius reprehenderit quasi totam velit! Voluptas illo hic incidunt nemo numquam est dicta pariatur alias optio, totam eius amet perferendis?

Таблицы

В различных расширениях языка Markdown, таблицы можно задавать символом |.

Пример:

| Клетка заголовка 1 | Клетка заголовка 2 | Клетка заголовка 3 | | ------------------ | ------------------ | ------------------- | | Клетка 1 | Клетка 2 | Клетка 3 | | Клетка 1 | Клетка 2 | Клетка 3 |

Приведет к генерации следующего HTML:

Клетка заголовка 1 Клетка заголовка 2 Клетка заголовка 3
Клетка 1 Клетка 2 Клетка 3
Клетка 1 Клетка 2 Клетка 3
